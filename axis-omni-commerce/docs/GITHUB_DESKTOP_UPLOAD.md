# Upload With GitHub Desktop

1. Extract `axis-omni-commerce-github-ready.zip`.
2. Open GitHub Desktop.
3. Choose **File → Add local repository**.
4. Select the extracted `axis-omni-commerce` folder.
5. When prompted, initialize it as a repository.
6. Use repository name: `axis-omni-commerce`.
7. Enter an initial commit message such as:

   ```text
   Initial AXIS OMNI commerce operating system
   ```

8. Select **Publish repository**.
9. Keep it private initially because it contains your operating framework.
10. Add collaborators only after deciding what parts should remain proprietary.
