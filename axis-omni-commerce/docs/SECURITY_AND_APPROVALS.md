# Security and Human Approvals

## Never commit

- API keys
- access tokens
- bank credentials
- payment processor secrets
- supplier portal passwords
- customer personal data
- unredacted contracts containing sensitive data

## Default production posture

- draft-only API permissions;
- least-privilege credentials;
- human approval before publication;
- human approval before spending;
- human approval before contracts or financial-account connections;
- full action logs;
- versioned rollback.

## Non-negotiable approvals

Contracts, debt, guarantees, inventory purchases, regulated claims, substantial ad spending, sensitive-data sharing, irreversible deletion, and unresolved legal questions require human approval.
